This algorithm simulates Naive Bayes classifier.
This can be tested with .dat files.

Compile:

javac NaiveBayesAlgorithm 

Run:

java NaiveBayesAlgorithm 1 0.2 false

1º arg - number of labeled file to test. Ex: 1 for 1-labeled.dat
2º arg - percentage of learning data. Ex: 0.2 for 20% of training data
3º arg - boolean value to show probabilities matrix. Ex: 1 for 1-labeled.dat
